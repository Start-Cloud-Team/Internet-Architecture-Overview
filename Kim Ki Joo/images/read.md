텍스트 편집기(VS Code, Sublime Text 등)로 페이지 제목.md 파일을 열고, 이미지 경로를 Git 저장소 기준으로 수정합니다.

Notion에서 내보낸 마크다운 파일의 이미지 경로는 보통 다음과 같습니다.

Markdown

![이미지 이름](페이지%20제목%20폴더/이미지%20파일.png)

Markdown

![이미지 이름](/assets/images/이미지 파일.png)
